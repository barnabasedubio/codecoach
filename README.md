# vue-coach-finder
Frontend coach finding app built with VueJS.

Try it out [here](https://barnabasedubio.github.io/codecoach/).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
